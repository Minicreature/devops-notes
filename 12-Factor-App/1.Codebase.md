# 1. Codebase

## Git
**Why git is used?**
Git is used to manage codebase, keep it in track & structured manner. Also for doing version control as well.

## GitHub
**Why github is used?**
To manage same codebase by multiple developers avoiding version conflicts, merge issues, etc & providing ease of doing it by handling multiple environments.

## Distributed Systems
Sharing same codebase for multiple apps/app services is violation of 12 Factor app rule. Each app should have its individual codebase.
